# QA Archive

Thư mục này chứa các báo cáo QA cũ đã được archive để làm gọn thư mục `docs/qa/`.

## Nguyên tắc

* Không xóa lịch sử QA nếu chưa có xác nhận.
* File được move vào đây để giảm nhiễu, không phải bị loại bỏ.
* Khi cần tra cứu, xem manifest chi tiết tại:
  * `docs/qa/QA_ARCHIVE_MANIFEST_2026_06.md`

## Nội dung archive

Tóm tắt các nhóm tài liệu chính đã được đưa vào archive:
* Các báo cáo QA cũ từ Phase 1 đến Phase 6.
* Các file log kiểm tra UI/UX, JSON test artifacts.
* Các báo cáo dọn dẹp dữ liệu, test data cleanup.
* Ảnh screenshot, log test tự động từ Playwright.
* Các xác nhận nghiệp vụ (RBAC, Documents, Payments...).
